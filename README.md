<h1>PhonesAPI</h1>
RESTful API built with ASP.NET Core 3 and Entity Framework Core (SQL Server).
Searching, filtering, pagination, sorting, data shaping, data caching, logging, versioning, DI, etc.
Documentation with Swagger / OpenAPI
